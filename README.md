# SSH Authentication Investigation (Kali Linux)

## Overview
This project simulates SSH login attempts and analyzes authentication activity using logs and packet capture.

## Tools Used
- Kali Linux
- OpenSSH
- tcpdump
- tshark (Wireshark CLI)

## Key Findings
- Multiple failed login attempts detected
- Invalid user "kali" used in attack attempts
- Successful login for user "ajit"
- SSH traffic captured on port 22

## Project Structure
- Logs → auth.log
- PCAP → SSH packet capture file
- Screenshots → proof of setup and analysis
- Report → detailed investigation report
